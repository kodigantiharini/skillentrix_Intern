# 🎬 Movie Recommendation System

## Project Overview
This project implements a Movie Recommendation System using Machine Learning techniques.

## Features
- Content-Based Filtering
- Collaborative Filtering (SVD)
- Hybrid Recommendation System
- Cosine Similarity
- RMSE Evaluation
- Precision@K
- Recall@K

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
An in-code sample movie dataset is used, so no external download is required.

## Algorithms
### Content-Based Filtering
Uses TF-IDF Vectorizer and Cosine Similarity to recommend similar movies.

### Collaborative Filtering
Uses Truncated SVD to predict user preferences.

### Hybrid Recommendation
Combines content-based and collaborative recommendations.

## Evaluation Metrics
- RMSE
- Precision@K
- Recall@K

## How to Run

```bash
pip install -r requirements.txt
```

Open the notebook:

```
jupyter notebook
```

Run all cells.

## Output
The notebook recommends movies based on movie genres and user ratings.
